## ⚙️ Getting Started

### ✅ Prerequisites

- Node.js (v18 or later recommended)
- npm or yarn

---

### 📦 Installation

1. **Clone the repository:**

```bash
git clone https://github.com/yourusername/portfolio.git
```

2. **Install dependencies:**

```bash
cd portfolio
npm install
```

---

### 🚀 Run Locally

```bash
npm run dev
```

Runs on: `http://localhost:5173`
